# 🧠 VLAExplain — Interpreting Vision-Language-Action (VLA) Models

VLAExplain is an interpretability toolkit designed to help users visually understand the inner workings of Vision-Language-Action (VLA) models.

Currently, attention analysis is supported for both the pi05 and unifolm-vla models.

For details, please check [pi05](src/lerobot/README.md) and [UnifoLM-VLA](src/unifolm-vla/README.md) readme files respectively.
